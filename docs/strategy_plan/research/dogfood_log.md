## Bug Severity Definitions

### Critical (P0) - Blocks usage
- **Definition**: Core workflow completely broken. Users cannot use Context.
- **Examples**: Agent doesn't respond, tasks don't sync to TickTick, login broken
- **SLA**: Fix within 24 hours

### High (P1) - Major friction
- **Definition**: Feature works but has serious issues. Workaround exists.
- **Examples**: Agent slow (>5 sec), wrong due dates, UI crashes on mobile
- **SLA**: Fix within 3 days

### Medium (P2) - Annoying but usable
- **Definition**: Feature has minor issues. Doesn't block usage.
- **Examples**: Typos, UI alignment, empty states missing
- **SLA**: Fix before beta launch (or post-launch if time-constrained)

### Low (P3) - Nice-to-have
- **Definition**: Polish issues. Not critical for beta.
- **Examples**: Keyboard shortcuts, dark mode, animations
- **SLA**: Post-launch backlog
